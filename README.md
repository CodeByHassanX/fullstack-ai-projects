## 📖 About This Repository

This repository serves as a growing portfolio of Machine Learning and Data Science projects. It covers a wide range of algorithms—from classical supervised and unsupervised learning models to fully deployed **Full-Stack AI Web Applications** (complete with React frontends and FastAPI backends).

Each project folder contains the datasets, Jupyter Notebooks for model training, and (for the later projects) the full source code to run the application locally.

### 🌐 Full-Stack AI Web Applications

These advanced projects take the trained models out of the Jupyter Notebook and deploy them into fully functional, interactive web applications featuring **FastAPI** backends and beautiful **React** frontends.

9. **[09. XGBoost: Customer Churn Prediction](./09_xgboost_customer_churn_prediction)**

11. **[11. K-Means: Customer Segmentation](./11_kmeans_customer_segmentation)**

## 🚀 How to Run the Full-Stack Apps

For Projects 10, 11, and 12, running the full application is incredibly easy.

1. Navigate to the specific project folder.
2. Double-click the **`run_app.bat`** file.
3. The script will automatically install all Python and Node.js dependencies, and start both the FastAPI backend and React frontend concurrently.
